Q21. What is a computer?

A computer is an electronic device that accepts data, processes it according to instructions, and produces output.

Q22.What are the main components of a computer system?

Input unit, Central Processing Unit (CPU), Output unit, and Memory.

Q23.What does CPU stand for?

Central Processing Unit.

Q24.Name the two main parts of CPU.

Arithmetic Logic Unit (ALU) and Control Unit (CU).

Q25.What is hardware?

Physical components of a computer that can be seen and touched.

Q26.What is software?

A set of programs or instructions that tell the computer what to do.

Q27.Name two types of software.

System software and Application software.

Q28.Give two examples of system software.

Windows, Linux.

Q29. Give two examples of application software.

MS Word, MS Excel.

Q30.What is an operating system?

Software that manages computer hardware and software resources.

Q31.What is input device?

A device used to enter data into the computer.

Q32.Give two examples of input devices

Keyboard, Mouse.

Q33.What is output device?

A device that displays or produces results from the computer.

Q34.Give two examples of output devices.

Monitor, Printer.

Q35.What is memory?

The storage area where data and instructions are stored.

Q36.What is primary memory?

Memory that is directly accessed by CPU.

Q37.Give two examples of primary memory.

RAM, ROM

Q38.What is RAM?

Random Access Memory; it is volatile memory.

Q39.What is ROM?

Read Only Memory; it is non-volatile memory.

Q40.What is secondary memory?

Memory used for permanent data storage.

Q41.Give two examples of secondary storage.

Hard disk, Pen drive.

Q42.What is a bit?

The smallest unit of data (0 or 1).

Q43.What is a byte?

A group of 8 bits.

Q44.What is data?

Raw facts and figures.

Q45.What is information?

Processed data that is meaningful.

Q46.What is a computer program?

A set of instructions given to a computer.

Q47.What is booting?

The process of starting a computer.

Q48.What is cold booting?

Starting the computer when power is off.

Q49.What is warm booting?

Restarting the computer without turning off power.

Q50.What is a desktop?

The main screen displayed after logging into the computer.

Q51.What is an icon?

A small graphical symbol representing a program or file.

Q52.What is a file?

A collection of related data stored under a name.

Q53.What is a folder?

A container used to store files.

Q54.What is GUI?

Graphical User Interface.

Q55.Give one example of GUI-based OS.

Windows

Q56.What is command-based interface?

Interface where commands are typed using keyboard.

Q57.Give one example of command-based OS.

MS-DOS

Q58.What is a peripheral device?

External devices connected to a computer.

Q59.Give two examples of peripheral devices.

Scanner, Printer.

Q60.What is a scanner?

An input device that converts paper documents into digital form.

Q61.What is a printer?

An output device that produces hard copy.

Q62.What is hard copy?

Printed output on paper.

Q63.What is soft copy?

Output displayed on screen.

Q64.What is storage capacity?

The amount of data a device can store.

Q66.What is a virus?

A harmful program that damages computer data.

Q66.What is antivirus?

Software used to detect and remove viruses.

Q67.What is a computer network?

A group of interconnected computers.

Q68.What is internet?

A worldwide network of computers.

Q69.What is email?

Electronic mail used to send messages over internet.

Q60.What is WWW?

World Wide Web; a collection of web pages on the internet.
